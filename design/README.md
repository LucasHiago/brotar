# Design

Espaço para identidade visual, fluxos e telas do app.

## A preencher
- Fluxos principais (telas): onboarding, lista de plantas, detalhe, pareamento do sensor.
- Paleta de cores e tipografia.
- Wireframes / mockups.
- Tom de voz das mensagens e recomendações (amigável, sem jargão técnico).

## Telas previstas no MVP
1. **Login / criar conta**
2. **Minhas plantas** — lista com status (verde / atenção / crítico)
3. **Detalhe da planta** — leituras atuais, gráfico de histórico, recomendações
4. **Adicionar planta** — apelido, foto, local, parear sensor
5. **Status do sensor** — bateria, sinal, última sincronização
